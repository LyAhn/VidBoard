# VidBoard

VidBoard is a local-first pre-production storyboard app for planning music video sequences from track details, lyrics, and visual direction.

## Prerequisites

- Node.js
- Ollama
- ComfyUI

## Run Locally

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Configure local services in `.env.local` using `.env.example` as a template.
3. Run the app:
   ```bash
   pnpm run dev
   ```
